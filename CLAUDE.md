# Kikiwake プロジェクト指示

## ファイルを読むタイミング
質問内容に関係するファイルだけを読むこと。全部読まない。

| 質問の内容 | 読むファイル |
|-----------|------------|
| データ保存・GAS・スプレッドシート関連 | `gas/コード.js` |
| 画面レイアウト・HTML構造関連 | `index.html` |
| ロジック・動作・バグ関連 | `script.js`（必要なら`index.html`も） |
| スタイル・見た目関連 | `style.css` |
| フロントとGASの連携関連 | `script.js` + `gas/コード.js` |
